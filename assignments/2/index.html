    Project 1: BMI Calculator

    Description:

    For this project we are asked to create a BMI Calculator. What is BMI? BMI is short for "Body Mass Index" and it weights a person usually in kilograms. We are asked to do the first set of code in imperial units. Once that is complete we are told to do another set of code using metric instead.

    Tech used:

    The Technologies used for this project is Intellij, an IDE that uses the java language

    Key Feature:

    The main key feature of this project is the third set of code, which is a combination of the previous sets of code. Also known as the "Master Version." This version allows the user to either do imperial or metric calculation.
